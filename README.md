# Idle game

A simple prototype for realisation basic idle game.

## Repository information

This repository contains an idle game demo scene without any graphics. In addition, in <a href="/GithubMedia">GithubMedia</a> you 
can find a calculation table for the economy of the game.

### Demo description

The demo scene is a set of clickable 3d objects. When you click on an object, its level increases (if there is enough money),
and consequently it starts to bring in more money.
